// 사용 언어 : React, HTML, CSS, 기타 언어

// 주제 : BGM 플레이 리스트 제작하기

//목표 : 자유자재로 재생 및 중지, 추가 가능한 뮤직 플레이 리스트를 만들기 입니다.

// 주요 기능:

노래 선택: 사용자가 재생할 BGM을 선택할 수 있도록 제공
재생/일시정지: 사용자가 선택한 BGM을 재생/일시정지할 수 있도록 제공
볼륨 조절: 사용자가 선택한 BGM의 볼륨을 조절할 수 있도록 제공
노래 변경: 사용자가 선택한 BGM을 변경할 수 있도록 제공
플레이 리스트 추가/삭제: 사용자가 자신의 플레이 리스트를 추가/삭제할 수 있도록 제공

// 개발 스택:

프론트엔드: React.js, HTML, CSS
백엔드: Node.js, Express.js

// 프로젝트 계획 :

4월 : 필요 기능 및 도구 선택, 기획 및 설계
5월 ~ 11월 :  Ul/UX 디자인, 코딩 구현(프론트엔드 개발, 백엔드 개발)
12월 : 통합 테스트 및 github

// 목적 및 기대 결과 : 

1 : 코딩 스킬 향상
    프로젝트를 통해 새로운 프레임워크를 습득하고 기존 코딩 실력을 개선해서 향상시키고, 
    또한, 프로젝트를 통해 여러 기술을 조합하여 새로운 솔루션을 만들어낼 수 있습니다.

2 : 문제 해결 능력 강화
    프로젝트를 진행하면서 생기는 문제들을 스스로 해결하면 문제 해결 능력을 강화할 수 있습니다.

4 : 경험 쌓기   
    프로젝트를 진행하면서 여러 가지 문제들을 경험하며 이를 해결하는 경험을 쌓을 수 있습니다.
    해당 경험은 현업 업무에 도움이 되고, 프로젝트 진행시 원할한 진행 가능
